Timestamp Microservice API

This timestamp API acccepts either a epoch/unix timestamp in seconds (integer) or a human-readale date string (GREGORIAN-MONTH DAY, YEAR) and return both formats in the form of a JSON object. Tested on Chrome, Firefox and Safari.

Example API Calls:

http://honmanyau-timestamp-api.glitch.me/819417600
http://honmanyau-timestamp-api.glitch.me/December 20, 1995
http://honmanyau-timestamp-api.glitch.me/December%2020,%201995

Example Output:

{"unix":819417600,"natural":"December 20, 1995"}
